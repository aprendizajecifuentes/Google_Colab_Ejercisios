# **Google Colab – Ejercicios de Análisis de Datos**

Este repositorio contiene ejercicios fundamentales de análisis de datos desarrollados por el ingeniero Juan David Cifuentes, utilizando Python, R y SQL.
Los ejemplos trabajan con datasets provenientes de Google Sheets y archivos tipo Excel, enfocados en técnicas de limpieza, transformación y análisis básico de datos.

## Contenido del Repositorio
### Procesamiento y Limpieza de Datos
  + *Normalización de fechas.*   
  + *Normalización y limpieza de texto.*
  + *Eliminación de duplicados.*
  + *Eliminacion Valores Nulos*
### Transformación y Agrupación
+ *Agrupación por columnas*
+ *Agrupación o combinación de tablas* (JOIN)     
### Análisis Aplicado
+ *Análisis de crédito*
+ *Simulaciones Motor Sobrecalentado*



## Herramientas Usadas
      
| Herramienta           | Uso principal                                |
| --------------------- | -------------------------------------------- |
| Python (Google Colab) | Procesamiento y análisis de datos            |
| R                     | Manipulación de datos y análisis descriptivo |
| SQL                   | Consultas y filtrado de información tabular  |
| Google Sheets / Excel | Origen de datos y preprocesamiento           |

## Cómo Utilizar los Notebooks

1. Abra cualquier archivo .ipynb desde Google Colab.
2. Si es necesario, conecte el notebook con Google Drive o cargue sus propios datasets.
3. Ejecute las celdas en orden para reproducir los ejercicios.
4. Modifique las funciones o agregue nuevas secciones según sus necesidades de análisis.
